Class: toucon-Page

toucon-Page

new toucon-Page(topContentopt, mainContentopt, bottomContentopt, bottomEqualsTopopt, useAuthorizationsopt, authorizedopt, useNotificationBaropt)

A container with three aggregations for content.
topContents will be added to a sap.ui.layout.HorizontalLayout
mainContents will be added to a sap.ui.layout.Grid
bottomContents will be added to a sap.ui.layout.HorizontalLayout
Parameters:
Name Type Attributes Description
topContent sap.ui.core.Control <optional>
 controls which are added here will be displayed horizontally
mainContent sap.ui.core.Control <optional>
 controls which are added here will be displayed in a responsive grid according to their layout data
bottomContent sap.ui.core.Control <optional>
 controls which are added here will be displayed horizontally
bottomEqualsTop boolean <optional>
 default=false; defines whether this page's bottomContent should be populated with a clone of the topContent (very useful for repeating buttons like Cancel or Save above & below a long form)
useAuthorizations boolean <optional>
 default=false; this will change the default value of authorized to false. This avoids that the page contents are visible until the authentication model returns false.
authorized boolean <optional>
 default=false; defines if the user is authorized to use this page; if false, then a box with a default error message will be displayed (very useful for passing the value of an authorization model)
useNotificationBar boolean <optional>
 default=true; defines whether this page's footer consists of a toucon.NotificationBar (hidden by default)
