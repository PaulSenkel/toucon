Class: toucon-Popup

toucon-Popup

new toucon-Popup(titleopt, iconopt, useNotificationBaropt)

The Popup control is sap.m.Dialog with a custom header which includes a (x) close button.
The CSS of this popup will automatically hide the title bar if the added content is a sap.m.Page.
Parameters:
Name Type Attributes Description
title string <optional>
 title of the Popup
icon string <optional>
 URL to an icon, e.g. sap-icon://home
useNotificationBar boolean <optional>
 default=true; defines whether this popup's subHeader consists of a toucon.NotificationBar (hidden by default)
