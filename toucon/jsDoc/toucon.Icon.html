Class: Icon

Icon

new Icon(iconopt, defaultIconopt, iconRulesopt, colorValueopt, colorRulesopt)

The Icon control enables us to display a colored icon based on incoming parameters which we then map to valid icons and valid colors
Parameters:
Name Type Attributes Description
icon string <optional>
 URL to an icon, e.g. sap-icon://home, OR a value which will be translated by the iconRules mapping
defaultIcon string <optional>
 URL to an icon, e.g. sap-icon://home, defaultValue is sap-icon://border, OR a value which will be translated by the iconRules mapping
iconRules object <optional>
 a map which translates the value passed in "icon" to a valid icon URL
colorValue string <optional>
 a web color, e.g. #ff0000, OR a value which will be translated by the colorRules mapping
colorRules object <optional>
 a map which translates the value passed in "colorValue" to a valid web color
