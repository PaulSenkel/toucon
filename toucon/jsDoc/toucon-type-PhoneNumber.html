Class: toucon-type-PhoneNumber

toucon-type-PhoneNumber

new toucon-type-PhoneNumber()

Checks if the passed string consists only of digits, brackets, forward slashes, spaces, minus and plus signs and allows for an optional "x" followed exclusively by digits; accepts zero-length strings if not mandatory.
