Class: toucon-FacebookButton

toucon-FacebookButton

new toucon-FacebookButton(linkopt, layoutTypeopt, actionTypeopt, showFacesopt, showShareopt)

This control enables you to embed a Facebook button into your UI5 app without having to worry about anything. This control creates the FB-ROOT div, imports the FB script and generates the button.
Parameters:
Name Type Attributes Description
link string <optional>
 The full link to the Facebook page you want to like or recommend.
layoutType string <optional>
 The way the button should look like, use "standard" for the original button the number of likes next to it, use "box_count" for a compact tile-like display including a bubble showing the number of likes, use "button_count" for a normal button showing the number of likes in a bubble on the right, use "button" for a simple button without the number of likes, default is "standard"
actionType string <optional>
 determines the name of the button, can be either "recommend" or "like" (default)
showFaces boolean <optional>
 whether faces of friends liking this page should be shown or not, default is false
showShare boolean <optional>
 whether a "Share" button should be shown in addition to the Like/Recommend button, default is false
