Class: toucon-ImageSlider

toucon-ImageSlider

new toucon-ImageSlider(autoPlayopt, autoResizeopt, lazyLoadingopt, slideDurationopt, numberOfThumbnailsopt, maxWidthopt, maxHeightopt, loadingGifUrlopt, imagesopt, thumbnailsopt)

This ImageSlider enables the integration of the JSSOR ImageSlider in UI5 Apps.
It has been optimized for the thumbnail-based slider demo'ed on http://www.jssor.com/demos/image-slider.html
You may have to extend it to cover other Jssor Sliders (additional parameters for feeding other options).
If you do so, please let me know so that I can update this component.

Credits for the Jssor Slider library go to JSSOR.COM, please go there for up-to-date code, license information and donations.

v1.01 - added parameter for CSS with auto-include function so that the jssor CSS does no longer need to be included in another CSS file
Parameters:
Name Type Attributes Description
autoPlay boolean <optional>
 determines if the slideshow will run automatically, default is false
autoResize boolean <optional>
 determines if the slideshow container will be resized via timeouts when the screen resizes or the screen orientation changes, default is true
lazyLoading int <optional>
 determines if lazyLoading is used and how many slides will be preloaded, default is 1 which means that lazyLoading is true and one slide will be preloaded
slideDuration int <optional>
 milliseconds before each slide change if autoPlay is active, default is 1000 (1s)
numberOfThumbnails int <optional>
 number of thumbnails which will be displayed at the button of the ImageSlider, default is 5
maxWidth int <optional>
 width in px, default is 600
maxHeight int <optional>
 height in px, default is 300
loadingGifUrl string <optional>
 URL to animated gif, default is res/toucon-extlibs/jssor/loading.gif
images Array.<string> <optional>
 array of image URLs
thumbnails Array.<string> <optional>
 array of thumbnail image URLs

(protected, static) initSlider()

Sets the options for the JssorSlider, binds the resize events and launches the initial scale so that the slider will have the optimal size right from the beginning.
Jssor Options:
$DragOrientation: 3 - you can drag left/right and top/down
$AutoPlay: from UI5 param
$LazyLoading: from UI5 param
$SlideDuration: from UI5 param
$ThumbnailNavigatorOptions : {
$Class : $JssorThumbnailNavigator$ - thumbnails are enforced
$ChanceToShow : 2 - thumbnails are always shown
$DisplayPieces : from UI5 param
}
