A text which can be indented and colored based on mapped rules

Methods

(protected, static) renderer(oRmopt, oControlopt)

Applies the params, resolves the values of indentLevel, color, background color against their respective indentRules and colorRules mappings, renders the text with the optional indentLevel multiplied with the indentBy parameter and the optional color and calls the sap.m.TextRenderer.render function.
Please note that the width will be automatically be set to 100% if
- the background color option is used
- text is aligned to left or center
