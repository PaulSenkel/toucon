Class: toucon-FormLine

toucon-FormLine

new toucon-FormLine(labelopt, tooltipTextopt, mandatoryopt, visibleopt, controlopt, labelLayoutDataopt, controlLayoutDataopt)

A container with two labels (one shown on medium and large screens, one on smartphones which takes a Control to which it attaches a richtext tooltip.
In the future we want to provide an additional label which displays an error text if an error is raised (e.g. through the change event based on the valueState or other information).

V1.01 - added visible flag
Parameters:
Name Type Attributes Description
label string <optional>
 used as "Title:" in front of the field, can be a {binding}
tooltipText string <optional>
 in a richtexttooltip which is being attached, can be a {binding}
mandatory boolean <optional>
 used for adding a styleclass; this will also be set to true if a toucon value type with a mandatory:true formatoption is being attached
visible boolean <optional>
 used for managing visibility of the labels and controls, default is true
control sap.ui.core.Control <optional>
 e.g. a sap.m.Input which should be rendered in the form line
labelLayoutData sap.ui.layout.GridData <optional>
 must contain the span for the label which will be shown on Large and Medium screens
controlLayoutData sap.ui.layout.GridData <optional>
 must contain the span for the control itself
