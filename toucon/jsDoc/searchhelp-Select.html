Class: searchhelp-Select

searchhelp-Select

new searchhelp-Select(serviceUrlopt, searchHelpIdopt, keyFieldopt, keyBindingPathopt, valueFieldopt, displayKeyopt, listModelopt)

The searchhelp.Select is a drop down field which creates its own List model based on the serviceUrl and searchHelpId unless the listModel is set externally.
Parameters:
Name Type Attributes Description
serviceUrl string <optional>
 the URL of the data service
searchHelpId string <optional>
 name of the backend value help
keyField string <optional>
 name of the SAP column whose data should be written back to the data model
keyBindingPath string <optional>
 path against which the selectedKey property will be bound
valueField string <optional>
 name of the SAP column from which the display value should be retrieved
displayKey boolean <optional>
 decides if the KEY will be combined with the VALUE for display as "KEY - VALUE"
listModel sap.ui.model.Model <optional>
 enables us to provide an external model instead of loading one from the serviceUrl if too many results have been returned (also used by the TableSelectDialog if the search still returns too many results.

Methods

(protected, static) buildModel(callBackopt)

Retrieves the attributes by calling [serviceUrl]/SearchHelpSet('[searchHelpId]')/AttributeSet?$select=Fieldname,ScrtextL,Datatype,Shlpinput,Shlpoutput,Shlpselpos,Shlplispos,Defaultval&$format=json
Then retrieves the values by calling [serviceUrl]/ValueSetBySearchString/?searchHelpName='[searchHelpId]'&searchString='[selectOptions]'?$select=Fieldname,Recordpos,Fieldval,Retfield&$format=json
Transforms the received ODATA-JSON to a flat JSON model and sets it as listModel.
Finally fires the modelBuilt event and executes the eventual callBack function.

(protected, static) triggerBinding()

Takes the listModel if it exists and applies it as named "List" model to this control.
Then prepares the key and text bindings in a sap.ui.core.Item template and binds the items to the control.
Finally binds the keyBindingPath to the selectedKey property and fires the bindingTriggered event.
