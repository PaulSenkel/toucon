Class: toucon-NotificationBar

toucon-NotificationBar

new toucon-NotificationBar()

The NotificationBar control can be used as a header or footer bar.
By default the toucon.Page uses a NotificationBar as footer.

(protected, static) init()

Applies the standard arguments to the sap.m.Bar prototype.
Initializes the button, the popover, the list and the text element.
The button gets a press event which opens the popover and is added as leftContent of this bar.
The list gets a StandardListItem as bind template and is added to the popover.
The text is added as middleContent to this bar.

(protected, static) renderer(oRmopt, oControlopt)

Creates a JSON model based on the messages array and adds it to the notification list.
Goes through the messages array and determines which icon has the biggest weight.
The result of this will determine the color and icon of the button which opens the popover.
The last message's description will then be used as the main text of the bar.
At the end the sap.m.BarRenderer.render is called.
